# GA
### 基本流程
- 编码
- 初始化种群pop
- 计算适应度
- 选择：轮盘概率选择，（主要是选几个出来进行交叉变异）
- 上一步pop进行交叉
- 上一步pop进行变异
- 上一步pop和原始pop合并，（子代和父代）
- 选择最好的几个pop，（这里是单纯选择好的保留）

### 重点
- 编码的设计
- 适应度计算设计
- 交叉和变异算子
